## Getting Started

Simply you can run this because no complex backend is added.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iamkaushalkb/BEI_CPROGRAMMING.git
   ```
2. Install GCC C comiler

3. Run Code
   ```sh
   gcc program_name.c -o program_name
   ```
